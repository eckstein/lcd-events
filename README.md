# LCD Events

A WordPress plugin for managing events for LCD.

## Features

- Event management
- Event calendar
- Event registration

## Installation

1. Upload the `lcd-events` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

## Requirements

- WordPress 5.0 or higher
- PHP 7.2 or higher
